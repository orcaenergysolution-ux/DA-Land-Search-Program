⚡ Transmission Line Property Finder

Finds land for sale close to high-voltage power lines. Set your criteria and press Search.

Search area
66 kV is the default. Tick more to include other line types.
Listing a few towns is the fastest way to search land parcels — it scans only the power lines near those towns instead of the whole state.
Property criteria
The search stops as soon as it finds this many, then shows the results. Keeps searches fast.
"Land" and "Rural" are cheapest to search and hold most big blocks. Add "House" to include acreage that already has a home on it.
Property type, listing type and price do not apply — the cadastre covers every block of land, whether or not it is for sale, so there is nothing to filter on.
Neighbours (privacy / isolation)
Checking neighbours uses a free public map service that is often slow or busy. It can add several minutes, and some results may come back unchecked. Turn it on only when you need it.
Counts buildings around the property using free map data. Adds about 1 second per matching property.
Data & cost
Scans along the power lines. It usually stops early once it has found the number of results you asked for, so this is just an upper limit. Always free.
